# Brain-Tumor-ML-Classifier
End to End real time brain tumor classification pipeline using TensorFlow/Keras transfer learning (MobileNetV2), TfLite conversion/optimisation and live inference with softmax confidence scoring. 

## Contents: 
- `brain-tumour-classification.ipynb`: Model development, training and evaluation (TensorFlow/Keras). Includes code descriptions.
- `tumor-inference.ipynb`: Inference script using OpenCV and code descriptions of inference.
- `classification_brain_tumor/tflite`: TensorFlow Lite model.

## Python Stack (Ecosystems/Libraries)
- tensorflow
- numpy
- opencv-python
- scikit-learn
- matplotlib
- pillow
